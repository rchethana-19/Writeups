# Cybersecurity Labs & Write-ups

This repository contains my hands-on cybersecurity labs, CTF write-ups, and blue team investigations completed through platforms like **TryHackMe** and personal practice.

The goal of this repository is to document my learning process, demonstrate practical skills, and build a portfolio of real-world security investigations.
